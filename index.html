建立hexo部落格

安裝Node.js、Git、Hexo

  • Node.js官網
  • Git官網
  • Hexo官網；安裝指令：npm install -g hexo-cli

確認安裝是否成功

  • node -v
  • npm -v
  • hexo -v

hexo 指令

  • 建立部落格：hexo init 你的網站名稱
  • 開啟伺服器：hexo server
  • 新增一篇部落格：hexo new “部落格標題”
  • 新增部落格頁面：hexo new page “頁面資料夾名稱”
  • 輸出實際網頁： hexo generate
  • 清除快取：hexo clean
  • 部署網站：hexo deploy
